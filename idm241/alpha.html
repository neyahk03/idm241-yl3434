Alpha Description

Link to Instacart

TRIGGER

"Add" button

  • Hover ON
  • Hover OFF
  • Click "Add to cart" button

RULES

"Add" button

Hover ON

  • Cursor changes to finger
  • Button width expands
  • "Add" changes to "Add to cart"
  • Filled background color changes to darker green

Hover OFF

  • Cursor changes back to pointer
  • Button width changes back to initial size
  • "Add to cart" changes back to "Add"
  • Filled background color changes back to lighter green

Click "Add to cart" button

  • Button changes to "1 ct"
  • Plus icon moves to the right
  • Trash icon appears on the left
