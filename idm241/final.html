RULES

Movie Tile - Default

  • Hover ON - cursor changes to finger
  • The card scales up by 10% and the shadow intensity increases
  • Hover OFF - cursor changes back to pointer
  • The card returns to its orignial size and shadow
  • Click ON - card flips to the back side

Movie Tile - Active

  • Hover ON - cursor changes to finger
  • The card scales up by 10% and the shadow intensity increases
  • Hover OFF - cursor changes back to pointer
  • The card returns to its orignial size and shadow
  • Click ON - card flips back to the front side

Play Button

  • Hover ON - button scales up by 10%
  • Hover OFF - button returns to its original size
  • Mouse DOWN - button scales down to its original size
  • Mouse UP - rediects to first episode of the movie

Bookmark Button - Default

  • Hover ON - button scales up by 10%
  • Hover OFF - button returns to its original size
  • Mouse DOWN - button scales down to its original size
  • Mouse UP - icon outline fills in with the color white

Bookmark Button - Active

  • Hover ON - button scales up by 10%
  • Hover OFF - button returns to its original size
  • Mouse DOWN - button scales down to its original size
  • Mouse UP - Icon filled-in to icon changes back to outline

Add To List Button

  • Hover ON - button scales up by 20%
  • Hover OFF - button returns to its original size
  • Mouse DOWN - button scales down to its original size
  • Mouse UP - opens model box

Close Button

  • Hover ON - cursor changes to finger
  • Button scales up by 20%
  • Hover OFF - curdor changes back to pointer
  • Button returns to its original size
  • Click ON - closes model box
